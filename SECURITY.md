# 安全策略

## 报告安全问题

如果你发现本项目存在安全漏洞或相关问题，请**不要**在公开 Issue 中报告。

请通过以下方式私下报告：

1. 在 GitHub 上创建 [Security Advisory](https://github.com/fwwdn/sensitive-stop-words/security/advisories/new)
2. 或通过 GitHub 私信联系项目维护者

我们将在收到报告后尽快回复并处理。

## 安全相关注意事项

本项目是一个纯文本词库集合，不包含可执行代码。但在使用时请注意：

- **输入验证**：在将词库集成到系统时，确保对读取的词条做适当的输入验证
- **编码安全**：读取文件时指定 UTF-8 编码，避免编码注入问题
- **更新策略**：建议定期更新词库以保持时效性
- **合规使用**：请确保你的使用场景符合当地法律法规

## 支持的版本

本项目为纯数据仓库，采用滚动更新方式，始终建议使用最新版本。
